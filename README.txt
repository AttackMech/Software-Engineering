Software Engineering

This course covers development of large-scale programming projects in organizations, including: requirements and analysis, specification design, coding, degubbing and testing, maintenance and throrough documentation. These topics are covered in a series of 3 assignments, which I have available here.

Assignments 1 & 2
	
	These assignments follow the development of a software program for a ficional company called Chocoholics Anonymous. It is basically a program to help people get fit and allow them to visit certified dieticians, nutritionists and more.

Assigment 3

	This assignment covers readings in software engineering using them to answer the question on the accuracy of cost estimations for software engineering projects.
